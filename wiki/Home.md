# Welcome to Jules Remediator Wiki

## 🚀 Vision

This repository is an **autonomous factory for Kubernetes error remediation**. Our "Remediator Pod" actively listens for FluxCD alerts, classifies the failure, and triggers **Jules** to automatically generate and apply fixes through a structured experiment-driven lifecycle.

## 🏗️ Core Pillars

1.  **Domain-Driven Design (DDD)**: Clear separation of concerns between core logic and infrastructure.
2.  **LLMOps Standard 2026**: Every remediation attempt is an experiment tracked in MLflow.
3.  **GitOps First**: All fixes follow FluxCD patterns and ensure cluster consistency.
4.  **Autonomous Reliability**: Closed-loop feedback system for continuous improvement.
